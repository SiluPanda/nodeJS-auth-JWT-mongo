## SET UP

Create a file `.env` and add values for following parameters

MONGO_URI=<>

PASSWORD_ENCRYPTION_KEY=<>

JWT_SEC=<>

EMAIL=<>

PASSWORD=<>

Install node dependencies

`npm i`

Start server

`node app.js`
